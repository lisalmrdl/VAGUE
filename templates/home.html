V.A.G.U.E.

Video-game Approximate Game-finding Utility Engine

A VAGUEly accurate search engine for games! Search our modest database of games using boolean, tf-idf, or a secret third search. Discover new games or look up old favorites.

HOW TO

Type any query in the search bar. Our smart search engine will determine which search function to use. Trigger boolean search by adding operators to your query:

  • AND: Results much match both search terms
  • OR: Results may match either search term
  • "": Exact phrase match
  • *: Wildcard, matches any number of characters (including zero)
