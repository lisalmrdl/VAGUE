SEMANA VI - Position y Pseudoelementos

Atributos de posiciones: position

Hay cinco posibles valores: static|absolute|fixed|relative|sticky|initial|inherit

Valor Descripcion
static Valor por defecto. Los elementos se renderizan en el mismo orden en el que aparecen en el c√≥digo
absolute El elemento se posiciona relativo a su primer elemento ascendente (ancestro, emparentado) que no tiene posicion est√°tica
fixed El elemento se posiciona relativo a la ventana del navegador
relative The element is positioned relative to its normal position, so "left:20px" adds 20 pixels to the element's LEFT position
sticky The element is positioned based on the user's scroll position
A sticky element toggles between relative and fixed, depending on the scroll position. It is positioned relative until a given offset position is met in the viewport - then it "sticks" in place (like position:fixed).
initial Sets this property to its default value.
inherit Inherits this property from its parent element.

Etiquetas multimedia

Las etiquetas multimedia se usan para introducir, valga la redundancia, archivos multimedia de audio, video, etc. Al igual que el resto de etiquetas tambien pueden llevar atributos, algunos exigen valores pero tambien pueden ser booleanos y que solo sea necesario poner el nombre del atributo, por ejemplo, "autoplay", "loop" o "controls". Las etiquetas multimedia requieren de una fuente o source (atributo src o con otra etiqueta dentro de la etiqueta correspondiente que sea "source").
